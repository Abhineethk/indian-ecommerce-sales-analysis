#  Indian E-Commerce Sales Analysis

## Project Overview

This project is an end-to-end Data Analytics project that analyzes e-commerce sales performance using SQL (MySQL) and Microsoft Excel. The objective is to identify revenue trends, top-performing products, category performance, state-wise and regional sales performance, monthly order trends, and order fulfillment efficiency through business-oriented SQL analysis and an interactive dashboard.

---

## Business Scenario

Assume you are a Data Analyst at an Indian e-commerce company similar to Flipkart or Meesho. Management wants to answer the following business questions:

- Which product categories generate the highest revenue?
- Which products contribute the most sales?
- Which states and regions generate the highest revenue?
- How do monthly order trends change over time?
- How effective is order fulfillment?

The original Brazilian Olist dataset has been adapted into an Indian business scenario for portfolio purposes by mapping Brazilian states to Indian states.

## Key Insights

- Health & Beauty generated the highest revenue among all product categories.
- Maharashtra generated the highest revenue among all states.
- West and South regions contributed the highest overall revenue.
- Monthly order volume increased steadily throughout the analysis period.
- More than 97% of orders were successfully delivered.
- Revenue was concentrated among a relatively small number of products and categories.
